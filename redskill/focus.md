## 评分逻辑（只在全部完成后执行）

```
维度分组：
  ATT: 题目 0-4
  IMP: 题目 5-9

每个维度计分：
  a（经常这样/是） = +1 分
  b（很少这样/否） = -1 分
  得分 > 0  → 偏向左边类型
  得分 < 0  → 偏向右边类型
  得分 = 0  → "均衡"

强度阈值：
  绝对值 >= 3 → "明显"
  绝对值 1-2  → "轻微"
  绝对值 = 0  → "均衡"
```

### 维度名称映射

| 维度 | 左类型 | 右类型 |
|------|--------|--------|
| ATT | 可维持型 | 易分散型 |
| IMP | 自控型 | 易冲动型 |

### 综合类型判断

| ATT 结果 | IMP 结果 | 综合类型 |
|-----------|----------|----------|
| 可维持型 | 自控型 | 专注稳定型 |
| 易分散型 | 自控型 | 注意力分散型 |
| 可维持型 | 易冲动型 | 冲动多动型 |
| 易分散型 | 易冲动型 | 混合型 |
| 其他组合 | 其他组合 | 专注稳定型 |

### 维度描述

**注意力维持 — 可维持型：** 大多数时候能专心完成任务，偶尔分心但能自己拉回来。
**注意力维持 — 易分散型：** 容易被外界干扰打断，做需要持续注意的事情时比较吃力。
**冲动控制 — 自控型：** 能管住自己，做事情前会想一想再行动。
**冲动控制 — 易冲动型：** 想到就做，很难先想再做。手脚停不下来，话也多。

### 训练方法

**可维持型训练：**
- 保持现有节奏，每天固定时间做需要专注的任务
- 用番茄法逐步延长专注时长（从15分钟开始）
- 完成任务后给予即时正向反馈
- 一次只做一件事，减少多任务切换

**易分散型训练：**
- 学习前清理桌面只留当前任务所需物品
- 用可视化计时器让他看见时间在流逝
- 把大任务拆成10-15分钟的小块
- 用身体锚定法——手里捏个软球反而能专心

**自控型训练：**
- 继续保持规则和边界清晰的环境
- 用暂停法——想说想做之前先数5秒
- 用角色扮演练习等待和轮流

**易冲动型训练：**
- 用暂停角——不是惩罚是冷静空间
- 教他"停·想·做"三步法——停一下想一想再做
- 多给身体活动机会释放多余能量
- 用闯关游戏的方式练习等待——每等1分钟积1分

### 家长建议

**可维持型家长：** 继续提供安静有序的学习环境；用计时器帮他看见自己的进步；不说"认真点"——说"你看你坚持了15分钟"
**易分散型家长：** 不要在他分心时吼——轻轻碰他肩膀拉回来；学习区远离窗户电视和走动的人；帮他找到消耗体力的方式——运动后专注力更好
**自控型家长：** 不用管太多——他自己节奏不错；如果偶尔冲动——提醒比惩罚有效；肯定他的自控力——"我刚才看到你忍住了"
**易冲动型家长：** 不公开批评他的冲动行为——私下提醒；提前预告——"5分钟后我们就要……"；运动是最好出口——每天保证户外活动时间
